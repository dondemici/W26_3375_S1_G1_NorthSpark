A personalized app where kids on the spectrum
create interest-based learning modules for
cognitive, emotional, and social skills

## Project Overview
**North Spark** is an innovative, dual-interface learning app designed for children on the autism spectrum, their parents, and therapists. By combining interest-driven flashcard creation, SEAT framework behavioral support, and real-time analytics, North Spark transforms traditional learning into personalized, engaging experiences.

**Motto:** *Learn what you love, your way.*

## Key Features
- **Child Interface:** Simple 3-7 card flashcard creator with streaks and sensory controls
- **Parent/Therapist Dashboard:** Session analytics, discrete trial tracking, IEP-ready reports
- **AI Recommendations:** Content adapts to special interests and excludes dislikes
- **SEAT Framework:** Addresses Sensory, Escape, Attention, and Tangible learning challenges
